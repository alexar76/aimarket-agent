# aimarket-agent

Reference consumer for **AIMarket Protocol v2** — discovery, payment channels, safety-gated invoke, settlement.

```bash
pip install -e ./aimarket-agent
aimarket-agent "translate spec to 5 langs + legal review" --base-url http://127.0.0.1:9080 --budget 3.0
```

See `aimarket-protocol/spec.md` and `docs/hub-integration-guide.md`.
